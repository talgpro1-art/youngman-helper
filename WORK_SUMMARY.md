# 지금까지 작업 요약

## 폐기한 방향

처음에는 국내 TOP100 차량 기반 AI 영업제안 레터 구독 서비스를 만들었습니다. 그러나 실제 뉴스 품질, 차종별 가격표 분석 정확도, 이메일 발송 운영 부담이 있어 공모전 시연용으로는 리스크가 크다고 판단했습니다.

## 새 방향

사이트 이름은 **영맨 헬퍼**입니다.

핵심 기능은 다음과 같습니다.

1. 상단에 국내 차량 판매 순위 TOP50 표시
2. 차량별 공식 가격표/PDF 링크 제공
3. 차량별 관심 옵션 이름과 기능 설명 제공
4. 관련 뉴스/신차/가격표 변경을 모니터링해 최상단 알림 제공
5. 모바일에서 보기 쉬운 카드형 UI 제공

## 현재 완성 상태

- Streamlit 앱 실행 가능
- 모바일 우선 UI 적용
- TOP50 차량 카드 표시
- 가격표/PDF 버튼 표시
- 옵션 설명 탭 표시
- notifications.json 기반 상단 알림 표시
- 뉴스 알림 갱신 스크립트 포함
- 공식 PDF 변경 체크 스크립트 포함
